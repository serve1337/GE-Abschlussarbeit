The Geometry type exposes the following members.
Methods
  NameDescription
AddVertex
Adds a vertex to the geometry container. Will return a handle to the newly inserted or still existing vertex.
CalcFaceNormal
This method adds a face normal vector to a list. The vector is calculated for the face which handle the method expects.
CalcVertexNormal
This method calculates vertex normals for a specific vertex in the geometry and inserts them at the corresponding half-edges on the correct faces. This method uses an angle based algorithm to determine whether to calculate with another faces normal or not.
CreateConnection
Establishes a connection between two vertices. 1) Creates two half-edges 2) Fills them with information 3) Creates an edge pointer container and adds it to the geo container. 4) returns a handle to an edge
EnAllEdges
Returns an enumerable of all edge handles in the geometry structure.
EnAllFaces
Returns an enumerable of all face handles in the geometry structure.
EnAllVertices
Returns an enumerable of all vertices handles in the geometry structure.
EnFaceAdjacentFaces
Iterator. Circulate around all the faces surrounding a specific face.
EnFaceAdjacentHalfEdges
Iterator. This is a method that retrieves all halfedge handles which belong to a specific face handle.
EnFaceAdjacentVertices
Iterator. Circulate around all the vertices of a given face handle.
EnStarVertexVertex
Iterator. Circulate around a given vertex and enumerate all vertices connected by a direct edge.
EnVertexAdjacentFaces
Iterator. Circulate around a given vertex and enumerate all faces adjacent to the center vertex.
EnVertexIncomingHalfEdge
Iterator. Circulate around a given vertex and enumerate all incoming halfedges.
EnVertexOutgoingHalfEdge
Iterator. Circulate around a given vertex and enumerate all outgoing halfedges.
